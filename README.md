# EmergencyForge Wiki

Das offizielle Wiki fuer EmergencyForge — gehostet ueber GitHub Pages mit MkDocs Material.

**Live:** [https://hypax.github.io/Wiki/](https://hypax.github.io/Wiki/)

## Lokal starten

```bash
pip install -r requirements.txt
mkdocs serve
```

## Beitragen

Jede Wiki-Seite hat einen Bearbeiten-Button, der direkt zur Datei auf GitHub fuehrt. Aenderungen koennen als Pull Request eingereicht werden.

Alle Inhalte liegen als Markdown-Dateien im `docs/`-Ordner.
